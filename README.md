# 🤖 WhatsApp Bot

A simple WhatsApp automation bot built using Node.js.

## 🚀 Features
- Auto reply to messages
- Simple command system

## 🛠️ Tech
- Node.js
- whatsapp-web.js

## ▶️ Run
npm install
node bot.js

## 💡 Commands
hi → Hello 👋  
help → Menu
